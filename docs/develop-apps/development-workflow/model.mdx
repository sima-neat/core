# Model

[`Model`](/reference/{lsa}/classes/simaai-neat-model) is the top-level API for loading a compiled model (`.tar.gz`) and exposing reusable stage fragments.

Use `Model` when you want model-aware graph assembly without manually wiring every model plugin.

## What `Model` gives you

- `graph()`: full model path as a reusable Graph fragment.
- `preprocess()`, `inference()`, `postprocess()`: reusable node-stage fragments.
- `input_specs()` and `output_specs()`: full tensor-contract lists for multi-input / multi-output models.
- `build(...)` / `run(...)`: direct convenience execution via `Model::Runner`.

Reference:
- [`Model` API](/reference/{lsa}/classes/simaai-neat-model)
- [`Model::Options`](/reference/{lsa}/structs/simaai-neat-model-options)
- [`Model::RouteOptions`](/reference/{lsa}/structs/simaai-neat-model-routeoptions)

## Typical usage inside a Graph

<CodeTabs>
<CodeTab label="C++" lang="cpp">

```cpp
simaai::neat::Model model("yolov8s_model.tar.gz");

simaai::neat::Graph graph;
graph.add(model.graph());
```

</CodeTab>
<CodeTab label="Python" lang="python">

```python
# Pseudo Python model example.
model = Model("yolov8s_model.tar.gz")

graph = Graph()
graph.add(model.graph())
```

</CodeTab>
</CodeTabs>

## Inspect specs and run directly

Before composing a Graph you can introspect the model's tensor contracts with `input_specs()` / `output_specs()` (and `metadata()`), then run a frame straight through with `run(...)` — no Graph required.

<CodeTabs>
<CodeTab label="C++" lang="cpp">

```cpp
simaai::neat::Model model("yolov8s_model.tar.gz");

// Tensor contracts the model expects (input) and produces (output).
const auto in  = model.input_specs().front();
const auto out = model.output_specs().front();
auto print_spec = [&](const char* label, const simaai::neat::TensorSpec& spec) {
  std::cout << label << " shape=[";
  for (std::size_t i = 0; i < spec.shape.size(); ++i) {
    std::cout << (i ? "," : "") << spec.shape[i];
  }
  std::cout << "]\n";
};
print_spec("input ", in);
print_spec("output", out);
std::cout << "metadata keys=" << model.metadata().size() << "\n";

// Run one frame directly via Model::Runner (no Graph required).
simaai::neat::TensorList result = model.run(std::vector<cv::Mat>{frame}, /*timeout_ms=*/2000);
```

</CodeTab>
<CodeTab label="Python" lang="python">

```python
model = pyneat.Model("yolov8s_model.tar.gz")

# Tensor contracts the model expects (input) and produces (output).
in_spec = model.input_specs()[0]
out_spec = model.output_specs()[0]
print("input  shape =", list(in_spec.shape))
print("output shape =", list(out_spec.shape))
print("metadata keys =", len(model.metadata()))

# Run one frame directly (no Graph required).
outputs = model.run([frame], timeout_ms=2000)
```

</CodeTab>
</CodeTabs>

## Failure handling

Model-driven Graphs use the same diagnostics contract as raw `Graph`:

- `NeatError.report().error_code` for terminal failures
- `NeatError.report().repro_note` for actionable context + hints
- `NeatError.report().bus` for plugin/runtime detail

Start triage from `error_code` (`misconfig.*`, `build.*`, `runtime.*`, `io.*`)
before inspecting detailed bus logs.
