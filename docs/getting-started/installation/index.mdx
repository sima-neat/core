As an AI application developer using the SiMa Neat framework, you have two options for setting up
your development environment. Both paths install the same framework and target the same DevKit
runtime. The main decision is where you want to do your day-to-day development work.

## Prerequisites

- You must have a SiMa.ai **Modalix 3.0+ Developer Kit**.
- We recommend creating an account at **[developer.sima.ai](https://developer.sima.ai)**
    - This will give you access assets such as precompiled models for testing.
- Install `sima-cli` using the [official guide](https://docs.sima.ai/pages/sima_cli/main.html) in both your DevKit and host environments.
    - `sima-cli` is used to install the Neat framework and to set up the Neat SDK / DevKit workflow.
