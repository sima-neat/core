You can install either the C++ or Python NEAT packages on the DevKit using the instructions below.

<CodeTabs>
<CodeTab label="C++" lang="cpp">
```cpp
# C++: install core from .deb and extras from .tar.gz.
# Download the latest artifacts from the release pages:
# - sima-neat-*-Linux-core.deb (core library and headers)
# - sima-neat-*-Linux-extras.tar.gz (examples, tutorials, and tests)

sudo apt install ./sima-neat-*-Linux-core.deb
mkdir -p "${HOME}/sima-neat-extras"
tar -xzf ./sima-neat-*-Linux-extras.tar.gz -C "${HOME}/sima-neat-extras"
```
</CodeTab>
<CodeTab label="Python" lang="python">
```python
# Python: install from PyPI.

pip install pyneat
```
</CodeTab>
</CodeTabs>

If you prefer building from source, follow the [build topic](./build.md).

Next, continue with [Hello SiMa](./minimal_example.md) to validate the install with a minimal app.
