## sima-cli

Install or update `sima-cli` on your Modalix DevKit before installing NEAT. You will use it both to install NEAT and to retrieve precompiled models for examples.

If you do not already have an account, sign up at [developer.sima.ai](https://developer.sima.ai).

Follow the official `sima-cli` installation guide:
[https://docs.sima.ai/pages/sima_cli/main.html](https://docs.sima.ai/pages/sima_cli/main.html)

After `sima-cli` is installed, choose one of the NEAT installation paths below.
Each path installs both the C++ and Python NEAT APIs on the Modalix DevKit.

Your Modalix DevKit must be connected to the Internet to install sima-cli and NEAT.

## Official Release

Recommended for most users.

```bash
cd /media/nvme && mkdir -p neat && cd neat && sima-cli install neat
```

You can optionally install extras, including prebuilt and source-code tutorials, examples, and tests.

## Latest Build

Use this to try the latest automated build.

```bash
wget -O /tmp/install-neat.sh https://tools.modalix.info/install-neat.sh && bash /tmp/install-neat.sh
```

## Specific Build

Use this to test a specific branch and commit.

```bash
sima-cli install -m https://neat-artifacts.modalix.info/neat/{branch}/{githash}/metadata.json
```

Replace:
- `{branch}` with your branch name
- `{githash}` with the 7-character Git commit hash

If you prefer building from source, follow the [build topic](./build.md).

The install also provisions PyNeat in `~/pyneat` virtual environment.
