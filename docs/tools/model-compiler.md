# Model Compiler

Model Compiler turns trained models into Modalix-ready model archives that Neat
can load and run.

Use it when you need to:

- check whether a model is compatible with the SiMa.ai platform;
- quantize and compile a supported model;
- validate accuracy and performance before using the archive in a Neat app.

For the task workflow, start with [Compile a Model](/compile-a-model/). For API
reference material, see [Model Compiler API](/reference/model-sdk-api/).
