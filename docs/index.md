# SiMa.ai Software Development

Welcome to the SiMa.ai software documentation. This site covers the full path of
building AI products on the SiMa.ai platform — from getting your environment set
up, to compiling a model, to developing and shipping an application with the
**Neat** framework.

<div class="overview-section-label">Where to start</div>

If you are new, work top to bottom: get set up in **Getting Started**, turn a
trained model into a SiMa artifact in **Compile a Model**, then build your
application in **Develop Apps**. **Tools** and **References** are there when you
need them.
